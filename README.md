## 一次進行多個檔案的格式轉檔

### 說明: 
此執行檔(.bat)為.wav 轉 .mp3 並僅限於此。
若要進行其他影音格式轉檔，請修改index.txt中的程式，並另存.bat檔即可。

### 轉檔步驟: 
1. 將我們的目標檔案存放至與 CV.bat、ffmpeg.exe 同個資料夾路徑之下 
2. 點兩下 CV.bat (執行檔會自動完成批次轉檔)
3. 將原始.WAV檔移入 Archive 資料夾
4. 重新命名轉檔完成後的目標檔案
5. 結束
